# Python CRUD Application

## 📌 Project Description

This is a simple CRUD (Create, Read, Update, Delete) application built using Python.
The project demonstrates basic database operations and backend logic implementation.

---

## 🚀 Features

* ➕ Create new records
* 📖 Read/View records
* ✏️ Update existing records
* ❌ Delete records

## ⚙️ Installation & Setup

### 1️⃣ Clone the repository

```bash
git clone https://github.com/your-username/your-repo-name.git
cd your-repo-name
```

### 2️⃣ Create virtual environment (Recommended)

```bash
python -m venv venv
```

Activate it:

* Windows:

```bash
venv\Scripts\activate
```

* Mac/Linux:

```bash
source venv/bin/activate
```

### 3️⃣ Install dependencies

```bash
pip install -r requirements.txt
```

### 4️⃣ Run the application

```bash
python app.py
```
